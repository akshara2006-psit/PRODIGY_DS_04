# PRODIGY_DS_04

## Task 04 - Sentiment Analysis

### Objective
Analyze and visualize sentiment patterns in social media data to understand public opinion toward different brands.

### Dataset
Twitter Sentiment Analysis Dataset

### Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- WordCloud

### Analysis Performed
- Data Cleaning
- Sentiment Distribution
- Top Mentioned Brands
- Brand-wise Sentiment Analysis
- Positive and Negative Word Clouds

### Outcome
Identified public sentiment trends and visualized how opinions differ across brands using exploratory data analysis techniques.
